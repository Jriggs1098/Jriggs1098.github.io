Visualization of High-Rank League of Legends Players

This data visualization was created from a dataset of over 30,000 people who play the game League of Legends, one of the most popular online multiplayer games in the world. Specifically, this dataset focuses on high-ranked players from each region based on their League Points (LP). LP is earned by players when they win a game, and then lost if they lose a game. This visualization revolves around determining whether or not any relationships exist in this dataset, such as between the number of games a person plays and their winrate or LP , or between the number of high-ranked players and region. By using filters, it is also possible to see what factors might affect the games played, LP, winrates, etc. The visulization below was created with HTML, CSS, JavaScript, and version 3 of the D3 JavaScript Library. The visualization were created using a modified version of this dataset.

The following functionality is provided in the visualization below:

  • Region Selection:
    • This provides an interactive map to select a specific region filter for the data.
    • Hover a region on the map or using the Legend to view information about that region in the bottom-left corner, and click on a region to view data for that region.
    • It should be noted that some regions, such as selecting "All Regions" have a very large amount of data, meaning without other filters applied it may affect the performance of the visualization.
  • Filter Selection:
    • This provides a set of filters that can be applied to the data, as well as a feature to reset the filters. The filters are described below:
    • Queue Type: Select the type of queue that should be applied to the data. In this case, different queues essentially act as different gamemodes.
    • Veteran Status: Select whether to show players who were high-ranked in previous ranked seasons, or if they are new to being highly ranked.
    • Promotion Status: Select whether to show players who recently reached a new Ranked Tier, or players who have been in their Ranked Tier for a while.
    • Ranked Tier: Select which Ranked Tier you would like to view in the data. Master tier is the lowest high-ranked tier and has no player limit. Grandmaster is the next highest tier that does have a player limit, but this varies based on the size of the region. Challenger is the highest tier and also has a player limit, generally the 200-300 based on the region's size. To reach Grandmaster and Challenger, players must first achieve a set LP-requirement, and can then promote into a higher tier if the player limit has not been reached.
  • League Points and Win Rate per Total Games Played:
    • This section provides an interactive data visualization based on the filters selected in the previous sections.
    • Each scatter chart shows the data colored by Ranked Tier as a circle. If the player has recently been on a "hotstreak" (winning at least 3 games in a row), they are shown as a square instead.
    • Hovering over a set of games in either chart shows the details for that specific section, including the game range, number of players, and the min, max, and average of that set of data.
    • Using the "Show Avg. Line" button shows a line on the respective chart representing the average value at each game interval. This changes based on filters or other interactions appplied.
    • The pie charts at the bototom of the page shows the breakdown of data for some aspects of the data, including the Ranked Tier, Veteran Status, and Promotion Status.
    • The data in these pie charts is affected by the filters applied. Hovering a game range in the scatter charts above briefly affects the data in these charts to represent the data that is being hovered. Hovering a section on the pie chart also temporarily affects the data in the scatter charts, including the Average Lines if applied.
